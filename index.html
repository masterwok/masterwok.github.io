Projects

Game of Life Screensaver (C#)

One of my most recent personal projects, this screensaver runs a wrapped version Conway's Game of Life. Everything from the color and size of the cells to the density of the seed generation is configurable. The algorithm I used is the optimized algorithm discussed in Michael Abrash's Graphics Programming Black Book.

Job Scheduling Emulation (Java)

This project emulates job scheduling in an operating system. Jobs can be added and removed from the system. The bar at the top of the application is a snapshot where jobs are located in memory. The dispatcher was implemented using the Highest response ratio next (HRRN) scheduling discipline. User documentation is available here.

Creature (Java)

The creature is part of a two-part project where students in my class created a game. The requirements were to create an animated character that would roam around an environment. This project uses JOGL, the OpenGL binding for Java.

Simple Game (Java)

This was the final project for my computer graphics course. It features student generated rooms, a mini-map, and multiple instances of the creature from the first part of the project. The goal of the game is to shoot down the creatures. Use the mouse to look around and shoot, and the keyboard to move. This project uses JOGL, the OpenGL binding for Java.

Connect Four (Java)

This is a console based wrap around version of Connect Four. The user plays against an AI opponent that is implemented using a game-tree along with α–β pruning. The difficultly of the opponent depends upon the depth of the game tree the user specifies at the beginning of the game.

Shortest Path (Java)

This project is my implementation of the A* search algorithm. This algorithm is used to find the shortest path possible while avoiding obsticles on a 2D grid. The X's represent high terrain while the O's represent the path found.
